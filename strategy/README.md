# Personal AI Public Stocks Strategy Framework

This folder defines a reusable investment framework for AI-related public equities. It is designed for manual use with AI assistance: point an AI tool at this folder, then ask it to analyze a company, review a portfolio, or update a watchlist using these rules.

This is an investing research framework, not financial advice. Every action should be reviewed against your own risk tolerance, liquidity needs, tax situation, and current market data.

## How To Use This Folder

1. Start with `core-methodology.md` to understand the L/T model.
2. Use `portfolio-construction.md` to decide target weights, position size, and rebalance actions.
3. Use `signals.md` to convert data and market behavior into green, yellow, orange, or red actions.
4. For a new company, copy `templates/company-research.md` into a company note and fill it out.
5. Each month, copy `templates/monthly-review.md` and run a portfolio review.

## Default Strategy

- Universe: AI-related public equities.
- Style: long-term structural core with tactical entry and exit overlay.
- Concentration: moderate, usually 8-15 holdings.
- Review cadence: monthly, plus event-driven reviews.
- Tactical signals: affect entries, trims, sizing, and exit timing, but do not replace the structural thesis.

## Folder Map

- `core-methodology.md`: definitions for structural change, L layers, T tiers, and scoring.
